# Templates

Reusable templates for ideation, evaluation, and demo structuring. Copy these into your use case folder and fill them in.

---

## Core Templates

| Template | Purpose | When to Use |
|----------|---------|-------------|
| [Use Case Canvas](use-case-canvas.md) | Structure brainstorm ideas into actionable use cases | Ideation phase — turn rough ideas into structured candidates |
| [Feasibility Scorecard](feasibility-scorecard.md) | Rate use case feasibility across 6 dimensions | Scoring phase — decide which use cases are buildable |
| [Demo Script Template](demo-script-template.md) | Structure a 5-minute demo presentation | Demo prep — create clear, time-boxed narratives |

## How to Use

### Use Case Canvas

Convert a rough idea into a structured business case. Copy into your use case folder, fill in each section during the brainstorm/ideation session.

**Answers:** What problem does this solve? Who benefits? What metrics show success?

### Feasibility Scorecard

Quantify feasibility so you can compare use cases objectively. Score across 6 dimensions: data availability, data quality, integration count, model complexity, visualization needs, and auth/security.

**Answers:** Can we build this in the time-box? Where are the risks?

### Demo Script Template

Ensure every demo tells a clear story within a 5-minute time box. Outline: problem (30s) → solution (60s) → live demo (2-3 min) → impact (30s) → the ask (30s).

**Answers:** What's the narrative? What does the audience see?
