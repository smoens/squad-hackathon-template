# Guides

Guides are step-by-step instructions for operating the hackathon framework. Start with **Getting Started**, then follow the suggested reading order below to run your first hackathon end-to-end.

---

## Suggested Reading Order

1. [Getting Started](getting-started.md) — prerequisites and setup
2. [Running Your First Hackathon](running-your-first-hackathon.md) — end-to-end walkthrough
3. [Running Ideation](running-ideation.md) — facilitate the brainstorm phase
4. [Demo Crafting](demo-crafting.md) — turn your PoC into a customer-ready demo

Then reference industry-specific and advanced guides as needed.

|----------|-------------|

| [Demo Crafting](demo-crafting.md) | Turn a PoC into a compelling customer-facing demo. |
